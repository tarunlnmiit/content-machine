# Data Science Worksheet Landing Page — Claude Design Prompt

Create a clean, professional landing page for a Data Science worksheet lead magnet using the brand kit below.

## Brand Kit

- **Background**: Dark Navy (#1a1a2e)
- **Text/Primary**: Cream (#c9c4b8)
- **Accent 1**: Muted Blue (#7896a8)
- **Accent 2**: Gold (#a89968)
- **Pop/CTA**: Coral (#e07856)

## Page Structure

### Hero Section (50% of page)
- Background: Dark Navy
- Headline (80pt, Cream): "Turn Your Model into Production SQL"
- Subheadline (32pt, Muted Blue): "Download the step-by-step checklist"
- Body text (16pt, Cream): "A practical guide to translating machine learning models into optimized SQL queries. Used by data engineers at Fortune 500 companies."

### Visual Element
- Centered icon/graphic: Abstract representation of model-to-SQL pipeline (use Muted Blue + Gold)
- Subtle background pattern: Vertical gradient from Dark Navy to slightly lighter (#2a2a3e)

### Form Section (40% of page)
- Background: Slightly lighter navy (#2a2a3e)
- Headline (40pt, Cream): "Get the Checklist"
- Body text (14pt, Accent 1): "7-page worksheet + bonus: common SQL translation mistakes"
- **[CONVERTKIT EMBED PLACEHOLDER]** — embed ConvertKit form here (form title: "Download Your Checklist", CTA button color: Coral)
- Form fields: Name + Email only

### Footer Section (10% of page)
- Background: Gold (#a89968)
- Text (14pt, Dark Navy): "By [Your Name] • Data Science insights for production ML • No spam, unsubscribe anytime"
- Minimal, left-aligned

## Design Specifications

- **Layout**: Single-column, centered, A4 or web-optimized
- **Typography**: Modern sans-serif (clean, accessible)
- **Spacing**: Generous padding (40px margins), breathing room between sections
- **Style**: Minimalist, professional, no clutter
- **Button**: Coral CTA button with rounded corners (8px), white text, 16pt bold
- **Accessibility**: High contrast ratios, readable on mobile and desktop

## Notes

- Lead with the transformation/benefit (model → SQL)
- Use data-driven language: "7-page worksheet", "used by Fortune 500"
- ConvertKit form should match brand colors where possible
- No distracting animations; clean, focused design
- Optimize for copy-paste into Canva or Claude Design
