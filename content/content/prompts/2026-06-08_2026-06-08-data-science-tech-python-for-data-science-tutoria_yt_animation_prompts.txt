# Animation Prompt — Title Card
## Context
Script: /Users/tarungupta/Making It Big/Claude/content-machine/content/scripts/2026-W24/2026-06-08_2026-06-08-data-science-tech-python-for-data-science-tutoria_yt.md
Episode: Your Analysis Doesn't Land Until Someone *Sees* It
Raw tag: [5-second title card — "Data Visualization & Storytelling with Python"]

## Task
Create a Remotion React component (`TitleCard.tsx`) that renders a 5-second animated title card (150 frames at 30 fps).

## Brand spec
- Show: Breath of Data Science
- FPS: 30
- Background: #1E1B2E
- Surface: rgba(255,255,255,0.05)
- Accent: #6B8FA8
- Accent2: #3D5F75
- Text: #f0f0f2
- Muted: rgba(240,240,242,0.50)
- Heading font: Space Grotesk
- Body font: Space Grotesk

## Text to display
"Data Visualization & Storytelling with Python"

## Animation spec
1. **0–8f**: Background fades in from transparent (opacity 0→1) over `#1E1B2E`.
2. **0–12f**: A horizontal accent line (2px, color `#6B8FA8`, width=60% of frame) slides in from left using `interpolate(frame, [0,12], ['-100%','0%'])` on `translateX`.
3. **6–20f**: Show name ("Breath of Data Science") fades and slides up (translateY 20→0, opacity 0→1) using spring physics (`fps`, `damping:80`, `stiffness:200`).
4. **14–30f**: Main title text fades and slides up with a 3-frame stagger per line if multi-line. Use spring.
5. **140–150f**: Entire composition fades to black (opacity 1→0).

## Code requirements
- Single `TitleCard` exported component
- Accept props: `{ titleText: string; showName: string; durationInFrames: number }`
- Use `useCurrentFrame`, `useVideoConfig`, `interpolate`, `spring` from `remotion`
- No external libraries beyond `remotion`
- Heading font: Space Grotesk

## Output
Full `TitleCard.tsx` file content, ready to drop into `remotion/src/compositions/`.
