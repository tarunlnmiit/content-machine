# Animation Prompt — Title Card
## Context
Script: /Users/tarungupta/Making It Big/Claude/content-machine/content/scripts/2026-06-03_2026-06-01-life-self-dev-the-cost-of-carrying-things-nobody_yt.md
Episode: The Cost of Carrying Things Nobody Sees
Raw tag: [3-second title card — "The Cost of Carrying Things Nobody Sees"]

## Task
Create a Remotion React component (`TitleCard.tsx`) that renders a 3-second animated title card (90 frames at 30 fps).

## Brand spec
- Show: Breath of Life
- FPS: 30
- Background: #0f0b08
- Surface: #1c1510
- Accent: #d4885a
- Accent2: #7eb896
- Text: #f2ece4
- Muted: #9b8d82
- Heading font: Georgia
- Body font: Inter

## Text to display
"The Cost of Carrying Things Nobody Sees"

## Animation spec
1. **0–8f**: Background fades in from transparent (opacity 0→1) over `#0f0b08`.
2. **0–12f**: A horizontal accent line (2px, color `#d4885a`, width=60% of frame) slides in from left using `interpolate(frame, [0,12], ['-100%','0%'])` on `translateX`.
3. **6–20f**: Show name ("Breath of Life") fades and slides up (translateY 20→0, opacity 0→1) using spring physics (`fps`, `damping:80`, `stiffness:200`).
4. **14–30f**: Main title text fades and slides up with a 3-frame stagger per line if multi-line. Use spring.
5. **80–90f**: Entire composition fades to black (opacity 1→0).

## Code requirements
- Single `TitleCard` exported component
- Accept props: `{ titleText: string; showName: string; durationInFrames: number }`
- Use `useCurrentFrame`, `useVideoConfig`, `interpolate`, `spring` from `remotion`
- No external libraries beyond `remotion`
- Heading font: Georgia

## Output
Full `TitleCard.tsx` file content, ready to drop into `remotion/src/compositions/`.
