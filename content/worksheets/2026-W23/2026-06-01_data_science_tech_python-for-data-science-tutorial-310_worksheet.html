Stop writing loops: your NumPy array thinking upgrade

A working ML engineer thinks in arrays, not iterations. Move through every section with your own data — vectorized thinking is a prerequisite for everything downstream: pandas, training loops, tensor ops, inference at scale.

Find your loop problem

Think of the last time your Python script felt slow — or you reached for a for loop to process numbers. Describe it briefly below.

What were you computing? A sum, an average, a transform, a filter? Be specific about the operation, not just the goal.

How many rows or values? Hundreds, thousands, millions? Scale changes which fixes matter.

What happened when you ran it — slow, ugly to read, hard to extend? The pain point is where vectorization pays off.

2
Rewrite one loop as a NumPy operation

Take a list of numbers you care about — exam scores, expenses, steps walked, stock prices. Put them in an array:

import numpy as np data = np.array([ ... ]) # your real values, here

Write the loop version of one calculation (e.g. multiply every value by 1.1). Then rewrite it as a single vectorized NumPy expression — data * 1.1.

What changed in how you described the operation? You stopped saying "for each one, do this" and started saying "do this to all of them."

Use your array from section 2. Compute and write the actual numbers you got:

Now add one outlier — a value far outside the rest — and recompute mean and std. How much did each shift?

What does that tell you about which stat to trust when your data has long tails? (Hint: this is why production ML logs medians and percentiles, not just means.)

Build a 2D array from your own data

Create a 3×4 matrix from real life — monthly scores, weekly metrics, anything with rows and columns. Then inspect its shape.

m = np.array([[..,..,..,..], [..,..,..,..], [..,..,..,..]]) m.shape # → (3, 4) m.ndim # → 2 m.mean(axis=1) # row means m.mean(axis=0) # column means

Write what each axis result means in context — not just the number. What does the row average represent? The column average?
