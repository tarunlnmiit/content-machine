# Life Systems Kit — Tracker Guide (companion moat #2)

## Provenance
- Source: Life & Self-Development course worksheets (L1, L2, L5, L6)
- Created: 2026-06-06
- **TEMPLATE — reusable trackers, not personal data.** Example rows are fabricated to show the shape; delete them before you start.
- Each tracker is one plain CSV: open it anywhere, import it to Google Sheets or Notion, fill it weekly.

## What's in the kit

| Tracker | Lesson | Grain | What it measures |
|---|---|---|---|
| `energy_audit_tracker.csv` | L1 — Where your energy goes | One row per day | Daily activities, end-of-day energy, what drained vs gave |
| `keystone_habit_designer.csv` | L2 — One keystone habit that sticks | One row per week | A single habit, its trigger, and days hit out of 7 |
| `weekly_review_template.csv` | L5 — A light weekly review loop | One row per week | Five life areas rated 1–5 plus the week's one action |
| `deep_block_planner.csv` | L6 — Attention in a distracting world | One row per week | One protected deep block and how well it held |

The kit pairs with the worksheets, not replaces them. The worksheet teaches the idea once; the tracker is where you run it week after week. What gets reviewed gets better. What doesn't, drifts.

## 1. Energy Audit Tracker (L1)

Log one row each day for a week, then read the week as a whole. Be honest — this is private. You're not grading yourself on what *should* be there.

| Column | What goes in it |
|---|---|
| `date` | The day's date |
| `day_of_week` | Monday … Sunday |
| `main_activities` | The 3–5 things that actually took your time |
| `end_of_day_energy` | One of: High / Medium / Low / Crashed |
| `emotional_tone` | One word for how the day felt |
| `energy_drain` | The activity that took the most out of you |
| `energy_giver` | The thing that gave energy back |
| `obligation_or_choice` | Was the day mostly Obligation or Choice? |
| `notes` | Anything worth remembering |

**How to read it:** after seven rows, look for the two drains that repeat and the givers you keep skipping. Pick one small change for next week — protect a giver, or cut one drain in half.

## 2. Keystone Habit Designer (L2)

One habit. One row per week for at least four weeks. Make it embarrassingly small — small and done beats ambitious and abandoned.

| Column | What goes in it |
|---|---|
| `week_number` | 1, 2, 3, 4 … |
| `keystone_habit` | The single behaviour you're building |
| `trigger_statement` | "After/Before I ___ , I will ___ for ___ min" |
| `duration_minutes` | How long the habit takes |
| `target_days` | Usually 7 |
| `days_completed` | How many days it actually happened |
| `blockers_encountered` | What got in the way |
| `adjustment_next_week` | The one tweak for next week |
| `status` | Your read on whether it's holding |

**The rule (from the worksheet):** hit 4 or more days in a week and the habit is working. Miss 4 or more days two weeks in a row and something structural needs to change — shrink the habit or move the trigger, don't just try harder.

## 3. Weekly Review Template (L5)

Ten minutes, once a week. Rate five areas, name what worked and what drained, pick one action. Keep the cadence light — a heavy review is a review you'll skip.

| Column | What goes in it |
|---|---|
| `week_of` | The week's start date |
| `energy_sleep_rating` … `self_regard_rating` | Each area, 1 = rough, 5 = solid |
| `energy_sleep_word` … `self_regard_word` | One word describing each area |
| `what_worked` | One thing that went well, even slightly |
| `what_drained` | The moment or pattern that cost the most |
| `one_key_action` | The single thing you'll change next week |

The five areas are Energy/Sleep, Study/Work, Relationships, Phone/Distractions, and Self-regard.

**One-time setup (keep this somewhere you'll see it):** pick the day, time, and place you'll review; choose three questions you'll always ask; name the trigger ("after Sunday dinner"); and decide how you'll know the review is done. That setup is what turns a good intention into a loop — it lives here in the guide, not as a column you re-type every week.

## 4. Deep-Block Planner (L6)

One protected block of focused time, one row per week. Design the environment, not the willpower — remove the fight before it starts.

| Column | What goes in it |
|---|---|
| `week_of` | The week's start date |
| `deep_block_task` | The one task the block is for |
| `start_time` | When it starts |
| `duration_minutes` | 30–45 to begin with |
| `location` | A specific place — desk, library, rooftop |
| `phone_setting` | on / off / another room |
| `pre_block_prep` | The one thing to set up beforehand |
| `days_protected` | How many days out of 7 the block held |
| `what_went_well` | What worked this week |
| `what_kept_breaking` | What broke the block |
| `adjustment_next_week` | The one change to try |
| `honest_rating` | Struggled badly / Managed a few days / Mostly consistent / Nailed it |

**How to use it:** one real hour beats a scattered day. Protect the same block at the same time until it stops needing willpower, then add a second.

## Import to Google Sheets or Notion

**Google Sheets:** File → Import → Upload → choose the CSV → "Insert new sheet" → Separator "Comma". Then delete the example rows and freeze the header row (View → Freeze → 1 row).

**Notion:** New page → `/table` (or import CSV directly via `···` → Merge with CSV). Turn `end_of_day_energy`, `status`, `phone_setting`, and `honest_rating` into Select properties so you can filter and see patterns at a glance.

Keep the masters in this folder as your clean starting point. Copy them per cohort or per quarter rather than editing the originals — that way every fresh start begins from the same shape.
